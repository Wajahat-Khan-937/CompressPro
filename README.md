# CompressPro
CompressPro is a cross-platform desktop application that implements the Huffman Coding algorithm for lossless data compression. Unlike many compression tools that work only on text, CompressPro processes files at the binary level, making it compatible with any file type – including PDFs, images, Word documents, and more.

Built with Python and CustomTkinter, it features a modern user interface with real-time progress bars, compression statistics, user authentication, operation history, and both dark/light themes. The project was developed as part of a Data Structures and Algorithms course to demonstrate the practical application of binary trees, priority queues, and greedy algorithms.
